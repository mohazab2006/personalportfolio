# Information Architecture

## Main navigation

- Projects
- Experience
- Stack
- Contact
- Resume

## Homepage section order

1. Hero
2. Featured Work
3. More Projects
4. Experience
5. Tech Stack
6. Contact
7. Snaggy AI entry point

## Hero requirements

Must communicate:

- name
- positioning
- short supporting statement
- CTA to projects
- CTA to resume

## Projects requirements

Featured Work:

- 4 featured projects
- KnowledgeMesh is the hero project
- 3 supporting project cards

More Projects:

- clean smaller card grid
- supports breadth without clutter

## Experience requirements

- timeline-based layout
- CIRA
- AmanahTech
- concise role descriptions
- optional tags
- no tiny images

## Stack requirements

Grouped by category, not percentages:

- Languages
- Frameworks
- Infra / DevOps
- Security
- AI / ML & data
- Tools

## Contact requirements

Simple and direct:

- email
- GitHub
- LinkedIn
- resume
- optional short line about role interest
