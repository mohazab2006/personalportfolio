# Project Strategy

## Featured Projects

There are 4 featured projects:

1. KnowledgeMesh
2. SentinelStack AI
3. PathToOffer
4. StudyNFlow

## Hierarchy

KnowledgeMesh is the hero featured project.

It should receive the largest visual emphasis.

The other 3 are supporting featured cards.

## Why

This creates hierarchy and prevents the featured section from feeling flat.

## Featured section structure

- section title: Featured Work
- short supporting subtitle
- one large hero project card
- one row/grid of 3 supporting project cards

## More Projects

There should be a More Projects section below Featured Work.

Purpose:

- show range
- preserve breadth
- avoid cluttering the featured section

## Project card rules

Every card should prioritize:

- title
- one-line value statement
- stack tags
- clear CTA

Avoid:

- long paragraphs
- too many tags
- weak screenshots
- too much equal-weight presentation
