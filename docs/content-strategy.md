# Content Strategy

## Core rule

Recruiters skim.

Content must be high-signal and easy to scan.

## Copy style

- short
- strong
- clear
- low fluff
- high signal

## Avoid

- long personal essays
- generic passion statements
- repetitive filler
- too many bullets
- overexplaining obvious things

## Hero copy

Keep it tight.

One strong identity line.

One strong supporting line.

## Project copy

For featured cards:

- title
- one-line description
- stack tags
- links

For the hero project:

- slightly more detail, but still concise

## Experience copy

Each role should have:

- title
- org
- date
- 1–2 strong highlights
- optional tags

## Tech stack copy

Only grouped stack items.

No self-ratings.

No percentages.

No “expert / intermediate” visuals.

## Contact copy

Direct and minimal.
