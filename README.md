## ✨ Features

### 🤖 **Snaggy — AI Chatbot Assistant**
- Built a real-time AI chatbot using **Groq (LLaMA 3.3 70B)** for ultra-low latency inference  
- Implements **context-aware responses** using structured system prompts and dynamic user input  
- Designed a **streaming chat interface** for responsive, real-time interaction  
- Integrated via **Next.js API routes** for full-stack request handling  
- Enables visitors to explore projects, skills, and experience conversationally  

---

### ⚡ **Performance & Architecture**
- Server-side rendering with **Next.js App Router**
- Optimized performance using:
  - Lazy loading  
  - Code splitting  
  - Image optimization  
- Achieved **90+ Lighthouse score** across all metrics  
- Efficient API routing for low-latency AI responses  

---

### 🎨 **Advanced UI/UX Engineering**
- Built interactive UI using **Framer Motion** and smooth transitions  
- Integrated **React Three Fiber + Three.js** for WebGL-based 3D elements  
- Implemented responsive, mobile-first design with **Tailwind CSS**  
- Smooth scrolling powered by **Lenis**  

---

### 📱 **Responsive Design**
- Fully responsive across all screen sizes  
- Mobile-first architecture  
- Optimized touch interactions  
- Progressive Web App (PWA) ready

## 🎯 System Design Overview

**Chat Flow:**
User Input → Frontend Chat UI → API Route → Groq API → Response Stream → UI Render

**Key Concepts Implemented:**
- Prompt engineering with system instructions  
- Context-aware conversational responses  
- Streaming UI updates for better UX  
- Serverless backend architecture
