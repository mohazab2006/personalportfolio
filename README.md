# Mohamed Azab - Personal Portfolio

A modern, interactive portfolio website built with Next.js, showcasing my projects and skills as a Computer Science student specializing in AI & Cybersecurity.

## Features

- Custom cursor with magnetic effects
- 3D animated elements
- Smooth page transitions
- Interactive project showcase
- Responsive design

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- React Three Fiber
